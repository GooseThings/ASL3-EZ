🔢 Functions Stanza [functions]
DTMF commands decoded from the node's local receiver. Format: DTMF = class,method — the * prefix is implied.
ℹ️ DTMF commands are entered without the leading * (funcchar). So 3 = ilink,3 means users dial *3<node>. See the COP Commands reference for all available commands.
Local Function Commands
DTMF KeyCommand (class,method or class,method,params)
🔗 Link Functions [link-functions]
DTMF commands decoded when accessed via a linked node.
Link Function Commands
DTMF KeyCommand
📞 Phone Functions [phone-functions]
DTMF commands decoded from telephone (autopatch) connections.
Phone Function Commands
DTMF KeyCommand
🎛️ Macros [macro]
Macros execute a series of DTMF functions in sequence. Format: macroN = *cmd1:*cmd2:*cmd3
ℹ️ Each macro entry is a number = colon-separated list of DTMF commands (including the * prefix). Example: 1 = *81:*82
Macro Definitions
Macro #Command Sequence (colon-separated DTMF strings)
🕐 Scheduler [schedule]
Schedule macros to run at specific times using cron-style syntax.
ℹ️ Format: macroN = minute hour dom month dow — uses cron syntax. * = any. Example: 2 = 00 00 * * * runs macro 2 at midnight daily.
Scheduled Events
Macro #MinuteHourDOMMonthDOW

Define courtesy tones and telemetry sound definitions. Format: name = |t(freq,freq2,ms,amplitude)...
