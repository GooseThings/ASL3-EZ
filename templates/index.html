ASL3 global node lookup and DNS settings
Node Settings

🔢 Functions [functions]
DTMF commands from the node's local receiver. Format: DTMF = class,method — * prefix implied.
ℹ️ Keys are entered without the leading *. So 3 = ilink,3 means users dial *3<node>.
Local Function Commands
DTMF KeyCommand
🔗 Link Functions [link-functions]
DTMF commands decoded when accessed via a linked node.
Link Function Commands
DTMF KeyCommand
📞 Phone Functions [phone-functions]
DTMF commands from telephone (autopatch) connections.
Phone Function Commands
DTMF KeyCommand
📜 Macros [macro]
Execute sequences of DTMF functions. Format: N = *cmd1:*cmd2
ℹ️ Each entry: macro number = colon-separated DTMF commands (include the * prefix). Example: 1 = *81:*82
Macro Definitions
Macro #Command Sequence (colon-separated)
🕐 Scheduler [schedule]
Schedule macros using cron-style syntax.
ℹ️ Format: macroN = minute hour dom month dow. Example: 2 = 00 00 * * * runs macro 2 at midnight.
Scheduled Events
Macro #MinuteHourDOMMonthDOW
